Converting User Input


  • Display Value – will display an alert box with the value from the input box
  • Display Value + 1 – will display an alert box with the value from the input box added to 1.
    Pay close attention to the output.
  • Display Value Converted + 1 – will display an alert box with the value from the input box added to 1.
    Notice the difference in values.
    Inspect the code to see why this is different.
